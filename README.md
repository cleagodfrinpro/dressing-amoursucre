# Dressing — Amour Sucré NewGen

Refonte UX d'un dressing / paper-doll inspiré d'Amour Sucré NewGen.

Site statique (HTML + images) hébergé via **GitHub Pages**.

## Lancer en local

Ouvrir `index.html` dans un navigateur, ou servir le dossier :

```bash
python -m http.server 8000
# puis ouvrir http://localhost:8000
```

## Structure

- `index.html` — l'application complète (HTML/CSS/JS en un seul fichier)
- `img/` — les visuels (catégories : robes, hauts, coiffures, accessoires…)
